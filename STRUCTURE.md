# Site Structure

Clean, organized folder structure for easy editing.

## Root Files
- **index.html** — Homepage. Links to all pages and sections.
- **README.md** — Project info.

## Folders

### `/pages/` — All content pages organized by category

- **`/pages/travel/`** — Travel & journey content
  - `travel-map.html` — Interactive travel map
  - `northern-lights.html` — Northern lights photography & stories
  - `flight-log.html` — Flight log & aviation experiences
  - `dive-log.html` — Dive log & underwater adventures

- **`/pages/work/`** — Tech & professional work
  - `ai-projects.html` — AI projects & experiments
  - `xr-videos.html` — XR & video projects
  - `reality-capture.html` — Reality capture work

- **`/pages/about/`** — Personal & contact
  - `contact.html` — Contact & connection
  - `volunteering.html` — Volunteering & community

- **`/pages/media/`** — Visual collections
  - `gallery.html` — Photography gallery

### `/css/` — Stylesheets
- **style.css** — Main stylesheet (Squarespace styles)

### `/images/` — Image assets
- Add images here and reference in pages

### `/js/` — JavaScript files
- Ready for any custom scripts

### `/ai-projects/`
- Contains media for ai-projects.html page

### `/universal/`
- Shared SVG and assets for Squarespace styling

---

## How to Edit

1. **Edit a page**: Open the HTML file in the appropriate `/pages/` folder
2. **Update navigation**: Change links in `index.html`
3. **Add images**: Place in `/images/` and reference with `<img src="../images/..."`
4. **Style changes**: Edit `/css/style.css`
